PicZeus Privacy Policy

Effective date: June 9, 2026

English 简体中文

English version

This Privacy Policy explains how the PicZeus Chrome extension handles information. PicZeus is designed for local image compression in the browser. The current version does not include user accounts, analytics, advertising trackers, payment collection, remote logging, or server upload for image compression.

Scope

This policy applies to the PicZeus Chrome extension and its extension pages, including the popup, workspace, options page, and the user-initiated right-click image workflow. It does not apply to third-party websites that you visit in Chrome.

Information PicZeus processes locally

  • Images and folders you explicitly select from your device are processed locally in the extension workspace using browser APIs.
  • If you use the right-click image action on a webpage, PicZeus fetches the selected image URL only after your explicit action, so the image can be shown and compressed in the extension workspace.
  • PicZeus stores local preferences such as language, theme, default preset, output format, quality, maximum dimension, metadata preference, and filename suffix in Chrome extension storage.

PicZeus does not ask for your name, address, account credentials, payment information, health information, location, or contact list.

How information is used

The information above is used only to provide PicZeus' single purpose: local image compression, preview, and download/export of optimized files.

PicZeus may read image metadata or image dimensions when your browser exposes that information during local processing. This is used to render previews, calculate size savings, and create the optimized output you requested.

Data collection and sharing

  • PicZeus does not sell user data.
  • PicZeus does not transfer user data to third parties.
  • PicZeus does not use user data for advertising, analytics, credit decisions, or purposes unrelated to image compression.
  • PicZeus does not upload selected local images to a remote server for compression in the current version.
  • PicZeus does not operate a backend service that receives your image files, compression queue, settings, or generated outputs.

Local storage and retention

PicZeus stores preferences in Chrome extension storage on your device so the extension can remember your language, theme, default preset, and compression settings. These preferences remain on your device until you change them, clear extension data, reset Chrome data, or uninstall the extension.

Image files selected for compression are kept in the active browser session only as needed to show the queue, previews, and results. If you clear the queue, close the workspace, reload the extension page, or uninstall the extension, PicZeus does not keep a separate remote copy.

Permissions

  • contextMenus: used to add the PicZeus right-click action for images.
  • downloads: used to let you download optimized files or ZIP exports that you create.
  • storage: used to remember local preferences.
  • Host permissions: used only for the user-initiated right-click image workflow, so PicZeus can fetch the selected image URL you chose.

Remote code

PicZeus does not use remote JavaScript or remote WebAssembly. The extension's UI and compression logic are packaged with the extension.

Your controls

  • You choose which local images or folders to add to PicZeus.
  • You choose whether to use the right-click image action on a webpage.
  • You can clear the compression queue inside the extension workspace.
  • You can remove PicZeus or clear its extension data through Chrome.

Children's privacy

PicZeus is a general-purpose image utility. It is not directed to children and does not knowingly collect personal information from children.

Changes to this policy

If PicZeus adds accounts, cloud services, analytics, payments, or any other feature that changes data handling, this policy will be updated before those changes are published.

Contact

For privacy questions about PicZeus, contact ripenaloe@gmail.com.

简体中文版本

本隐私政策说明 PicZeus Chrome 扩展如何处理信息。PicZeus 设计用于在浏览器本地压缩图片。 当前版本不包含用户账号、分析统计、广告追踪、支付收款、远程日志，也不会为了图片压缩把本地选择的图片上传到服务器。

适用范围

本政策适用于 PicZeus Chrome 扩展及其扩展页面，包括弹窗、工作区、设置页，以及用户主动触发的网页图片右键流程。 本政策不适用于你在 Chrome 中访问的第三方网站。

PicZeus 在本地处理的信息

  • 你主动从设备中选择的图片和文件夹，会通过浏览器 API 在扩展工作区本地处理。
  • 如果你在网页图片上使用右键 PicZeus 操作，PicZeus 只会在你明确触发该操作后获取所选图片 URL， 用于在扩展工作区展示并压缩该图片。
  • PicZeus 会在 Chrome 扩展存储中保存本地偏好设置，例如语言、主题、默认预设、输出格式、 质量、最大尺寸、元数据偏好和文件名后缀。

PicZeus 不会要求你提供姓名、地址、账号凭据、支付信息、健康信息、位置或通讯录。

信息用途

上述信息仅用于实现 PicZeus 的单一用途：本地图片压缩、压缩前后预览，以及下载或导出优化后的文件。

当浏览器在本地处理过程中提供图片元数据或尺寸信息时，PicZeus 可能读取这些信息，用于生成预览、计算节省体积， 以及创建你请求的优化输出文件。

数据收集与共享

  • PicZeus 不出售用户数据。
  • PicZeus 不向第三方转移用户数据。
  • PicZeus 不会将用户数据用于广告、分析、信用评估或与图片压缩无关的目的。
  • 当前版本不会为了压缩图片而把你选择的本地图片上传到远程服务器。
  • PicZeus 当前不运营接收你的图片文件、压缩队列、设置或输出文件的后端服务。

本地存储与保留

PicZeus 会在你的设备上的 Chrome 扩展存储中保存偏好设置，以便记住语言、主题、默认预设和压缩设置。 这些偏好会保留在本地，直到你修改设置、清除扩展数据、重置 Chrome 数据或卸载扩展。

被选择用于压缩的图片文件只会在当前浏览器会话中用于显示队列、预览和结果。你清空队列、关闭工作区、重新加载扩展页面或卸载扩展后， PicZeus 不会保留单独的远程副本。

权限说明

  • contextMenus：用于添加 PicZeus 图片右键操作。
  • downloads：用于下载你创建的优化图片或 ZIP 导出文件。
  • storage：用于保存本地偏好设置。
  • Host permissions：仅用于用户主动触发的网页图片右键流程， 以便 PicZeus 获取你选择的图片 URL。

远程代码

PicZeus 不使用远程 JavaScript 或远程 WebAssembly。扩展的界面和压缩逻辑均随扩展包一起提供。

你的控制权

  • 你可以自行选择哪些本地图片或文件夹加入 PicZeus。
  • 你可以自行决定是否在网页图片上使用右键 PicZeus 操作。
  • 你可以在扩展工作区中清空压缩队列。
  • 你可以通过 Chrome 移除 PicZeus 或清除其扩展数据。

儿童隐私

PicZeus 是通用图片工具，不面向儿童，也不会有意收集儿童个人信息。

政策更新

如果 PicZeus 后续加入账号、云服务、分析、支付或其他会改变数据处理方式的功能， 本政策会在相关变更发布前更新。

联系方式

如有 PicZeus 隐私相关问题，请联系 ripenaloe@gmail.com。
